Сделать менеджер задач в джанго
1. Пользователь с правами просмотра и редактирования всех событий по названию Руководитель
1.1 Руководитель может посавить задачу любому пользователю
1.2 Руководитель может заврешить задачу у другого пользователя
1.3 Руквоводитель может передать задачу от одного пользователя другому
1.4 Рукводитель может добавить исполнителя задачи

2. Исполнитель задачи с доступом: 
2.1 Поставить задачу пользователю из списка разрешённых пользователей для постановки этой задачи
2.2 Получить задачу от другого пользователя. 
2.3 Завершить поставленную пользователю задачу. 
2.4 Передать поставленную пользователю задачу другому пользователю. 

3 Интерфейс руководителя отображает все задачи всех пользователей. 
3.1 Интерфейс руководителя добавить фильтр пользовател для выбора списка исполнителей задач
3.2 Интерфей постановки задачи исполнителю
3.3 интерфейс редактирования задачи
